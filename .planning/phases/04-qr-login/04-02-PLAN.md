**轮询定时器**:
- self.poll_timer = QTimer(self)，interval 1000ms，连接到 _do_poll
- self.expiry_timer = QTimer(self)，singleShot=True，interval 60000ms，连接到 _on_expired
- self._uni_id = ""
- self._consecutive_errors = 0
- 最大连续错误次数: _MAX_POLL_ERRORS = 3

**start_qr_flow(self)**:
1. 停止现有定时器
2. 隐藏 overlay
3. status_label 设置 "正在获取二维码..."
4. 创建临时 Pan123 实例调用 qr_generate:
   ```python
   try:
       pan_temp = Pan123(readfile=True, user_name="", password="")
       data = pan_temp.qr_generate()
   except Exception as e:
       logger.error(f"获取二维码失败: {e}")
       self.status_label.setText("获取二维码失败，请重试")
       self._show_expired_overlay()
       return
   ```
5. 保存 self._uni_id = data["uniID"]，self._pan_temp = pan_temp
6. 生成二维码图片:
   ```python
   import qrcode
   from PIL.ImageQt import ImageQt
   qr_content = data["url"] + "?uniID=" + data["uniID"]
   qr_img = qrcode.make(qr_content, box_size=5, border=2)
   qt_image = ImageQt(qr_img.convert("RGB"))
   pixmap = QPixmap.fromImage(QImage(qt_image))
   self.qr_label.setPixmap(pixmap.scaled(200, 200, Qt.AspectRatioMode.KeepAspectRatio, Qt.TransformationMode.SmoothTransformation))
   ```
7. status_label 设置 "请使用 123云盘 App 扫码"
8. 重置 self._consecutive_errors = 0
9. 启动 poll_timer.start() 和 expiry_timer.start(60000)

**stop_polling(self)**:
- poll_timer.stop()
- expiry_timer.stop()

**_do_poll(self)**:
1. 调用 self._pan_temp.qr_poll(self._uni_id)
2. try/except 网络异常:
   - self._consecutive_errors += 1
   - 如果 < _MAX_POLL_ERRORS: return（静默重试）
   - 如果 >= _MAX_POLL_ERRORS: stop_polling(), status_label 设置 "网络异常，请检查后重试" + 红色 (#CF222E)，_show_expired_overlay()
   - return
3. 成功时重置 self._consecutive_errors = 0
4. 根据 loginStatus:
   - 0 (等待扫码): 无需操作
   - 1 (已扫码):
     - status_label 设置 "扫码成功，请在手机上确认"
     - _show_scanned_overlay()
   - 2 (已确认):
     - stop_polling()
     - status_label 设置 "登录成功"
     - token = result["token"]
     - 构造 Pan123 并验证:
       ```python
       pan = Pan123(readfile=True, user_name="", password="", authorization="Bearer " + token)
       user_data = pan.user_info()
       if user_data is None:
           self.status_label.setText("登录验证失败，请重试")
           self._show_expired_overlay()
           return
       self.loginSuccess.emit(pan)
       ```

**_on_expired(self)**:
- stop_polling()（只停轮询，不停 expiry 因为已是 singleShot）
- poll_timer.stop()
- 自动重新开始: start_qr_flow()（60秒超时自动刷新，CONTEXT.md 决策）
